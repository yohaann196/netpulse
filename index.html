about netpulse

Netpulse is a crowdsourced service status tracker for 100+ popular online services — social networks, streaming platforms, developer tools, cloud providers, and more.

how it works

When a service goes down, people file a GitHub Issue with the matching label. Netpulse fetches those issues directly from the GitHub API and shows you which services have reports in real time — no backend, no scraping, no API keys required.

status levels
Operational0 open reports Degraded1–3 open reports Major Outage4+ open reports
seen something broken?

Hit ↑ report outage on any service card (or in the header) to file a GitHub Issue. It shows up here automatically within seconds.

open source

Netpulse is fully open source. Fork it, self-host it, or suggest new services.
